## 🍰 소개
클라이언트 개발자 용현정입니다.  
사람들에게 즐거움과 편리함을 제공하고자하는 마음에 개발자가 되었습니다.  
언제나 핵심적이고 간결한 것을 추구하며 일합니다.  
혼자 일하는 것보다 함께 성장하며 일하는 것을 좋아합니다.  
서로 의견을 주고받는 중에 일어나는 시너지 효과가 중요하다고 생각합니다.  
Unity와 C#을 사용하여 게임개발 및 VR개발을 합니다.  
최근 관심사는 [유니티 최신기능]입니다.  
감사합니다!
